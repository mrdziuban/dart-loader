# Dart Webpack loader example

This is a simple Hello World example of the [Dart Webpack loader](https://www.npmjs.com/package/dart-loader).

### Install dependencies

```bash
$ npm install
$ pub get
```

### Compile

```bash
$ npm run compile
```

### Serve

Run

```bash
$ npm run serve
```

then open http://localhost:8080 in your browser.
